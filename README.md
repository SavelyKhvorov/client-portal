# Vue 3 + Vite

# 1. Клонирование репозитория
git clone https://github.com/ваш-username/название-репозитория.git

# 2. Установка зависимостей
Если используете npm:
npm install

Если используете yarn:
yarn install

# 3. Запуск проекта в режиме разработки
Если используете npm:
npm run dev 

Если используете yarn:
yarn dev
